# PokéAgents Image Assets

Place your image assets in this directory:

- `pokeball.png` - Splash screen pokéball
- `fire_icon.png` - Fire type icon
- `water_icon.png` - Water type icon
- `electric_icon.png` - Electric type icon
- `psychic_icon.png` - Psychic type icon
- `grass_icon.png` - Grass type icon
- `ice_icon.png` - Ice type icon
- `logo.png` - App logo

You can generate these using:
1. AI Image generators (Stable Diffusion, DALL-E)
2. Design tools (Figma, Canva)
3. Pokémon sprite resources (for inspiration)

For the hackathon demo, the app uses placeholder icons from Material Icons.

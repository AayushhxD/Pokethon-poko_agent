# PokéAgents Lottie Animations

Place your Lottie animation files (.json) in this directory:

Recommended animations:
- `evolution.json` - Sparkling evolution animation
- `battle_attack.json` - Battle attack effects
- `pokeball_open.json` - Pokéball opening animation
- `victory.json` - Victory celebration
- `loading.json` - Loading spinner

Download free Lottie animations from:
- https://lottiefiles.com/
- https://lordicon.com/

Search for keywords like:
- "pokemon"
- "sparkle"
- "battle"
- "evolution"
- "magic"

For the hackathon demo, the app uses built-in Flutter animations.
